# Crawling-Datapack
A Minecraft datapack that allows you to crawl anywhere, anytime.

## How to use:

- To crawl, look down and sneak.
- To stand up, press space or look up.
- To toggle crawling on/off for yourself, type /trigger toggleCrawl
